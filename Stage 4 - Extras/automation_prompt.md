# Automatic Test Assignment

## Objective
Perform automatic testing of the trip planning functionality.

## Test Scenario
Plan a trip from **Göteborg** to **Paris** with the following settings:

- POI type: `Restaurant`
- Maximum distance to POI: `200m`

## Verification
Verify that:

- The route is successfully calculated
- The route contains several stops along the way

URL: https://chargeatlas.testscouts.se
